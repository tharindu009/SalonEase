### General Information
SalonEase is a salon appointment booking website designed to simplify scheduling for both clients and salon professionals. With an responsive interface, users can book their favorite services with ease. Salon owners can manage appointments, ensuring a smooth and hassle-free experience.

### Technologies
- frontend: React
- backend: Express JS, Node JS
- Database: MongoDB
